## ▶️ How to Run Any Project

### Prerequisites
- Java 21 LTS installed ([Download Temurin](https://adoptium.net/))
- IntelliJ IDEA (Community Edition)
- MySQL 8.0 (for JDBC/intermediate projects)
- Maven 3.9+ (for Spring Boot projects)

### Steps

```bash
# 1. Clone this repository
git clone https://github.com/Aartigore14/java-mini-projects.git

# 2. Navigate to any project folder
cd java-mini-projects/01-basic/calculator

# 3. Open in IntelliJ IDEA
# File → Open → select the project folder

# 4. Run Main.java
# Right-click Main.java → Run 'Main'
```

> For JDBC projects, create the required MySQL database first.  
> Each project folder has its own `README.md` with specific setup steps.
